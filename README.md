# MovieLens iALS Recommender System

Проект представляет собой систему рекомендаций фильмов на основе датасета MovieLens 1M. Основная цель — реализация полного цикла ML-пайплайна от разведочного анализа данных до инференса рекомендаций с использованием алгоритма матричной факторизации iALS.

## Архитектура и подходы

В рамках проекта реализованы следующие этапы:
1. **Предобработка (Cold Start)**: Отфильтрованы неактивные пользователи и непопулярные фильмы, чтобы сгладить проблему холодного старта.
2. **Валидация (Per-user time-based split)**: Вместо глобального отсечения по времени реализован корректный per-user сплит. В тестовую выборку отложены последние взаимодействия для каждого пользователя.
3. **Моделирование**: Обучена модель на базе `AlternatingLeastSquares` из библиотеки `implicit`. Перевод данных осуществлен через разреженные матрицы.
4. **Метрики качества**: Реализован самописный расчет метрик ранжирования NDCG@K и Recall@K для честной оценки рекомендаций новых фильмов.

## Структура репозитория

- `data/` — директория для датасета (добавлена в .gitignore).
- `notebooks/main_model_experiments.ipynb` — черновик с EDA, визуализациями и прототипированием логики.
- `src/` — исходный код модулей:
  - `data_prep.py` — загрузка, сплит и фильтрация данных.
  - `model.py` — класс-обертка модели с методами fit/predict.
  - `metrics.py` — логика расчета NDCG и Recall.
  - `inference.py` — функции для получения читаемых рекомендаций.
- `main.py` — единая точка входа для запуска всего пайплайна.

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
3. Запустите пайплайн:
   ```bash
   python main.py
